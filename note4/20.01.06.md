### GraphQL introspection
其实是查询
`__schema`
`__type`

查看有哪些mutation类型
```js
{
  __type(name: "Mutation") {
    name
    fields {
      name
      type {
        name
        fields {
          name
        }
      }
    }
  }
}
```

关键词：
`fields` `name`


基本组件
```js
type Character {
  name: String!
  appearsIn: [Episode!]!
}
```
