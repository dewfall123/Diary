### git
git checkout -b [branch]
新建分支并切换



### push
git push [remote] [branch] // 远程仓库名 分支

### git status
- 修改的文件
- 未跟踪的文件
- 与服务区偏离的文件

git status -s // 简略版

### glob模式
`*` 表示任意字符
`**` 表示任意目录



### git diff
git diff // 比较工作区与暂存区的不同
git diff -staged // 比较暂存区与本地仓库的不同


### git commit
`- m` 添加注释
`- a` 直接 工作区 -> 本地仓库


### git log
`git log -p` 

### remote
clone 默认的远程仓库名是origin
`git remote add <shortname> <url>` 


### branch
`git branch home` // 新建分支
`git checkout -b home`
